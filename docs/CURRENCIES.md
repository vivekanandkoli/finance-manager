## How Currency Conversion Works

### Automatic Conversion
All amounts are automatically converted to INR (base currency) for:
- **Total calculations** across accounts
- **Dashboard summaries**
- **Analytics and reports**
- **Budget tracking**

### Display Format
- **Individual Items**: Shows in original currency
  - Account: "US Bank: $5,000"
  - Income: "Salary: $3,000"
  
- **Totals**: Shows in INR
  - Total Assets: ₹8,30,000
  - Total Income: ₹2,50,000

### Exchange Rates
- **Built-in rates** are approximate and used for calculations
- **Real-time updates** (coming soon)
- **Manual override** option (future feature)

## FAQ

### Q: Can I change the base currency from INR?
**A:** Currently INR is the base currency. Future updates will allow changing the base currency.

### Q: How accurate are the exchange rates?
**A:** Rates are approximate and updated periodically. For precise calculations, use real-time rates from your bank.

### Q: Can I track cryptocurrency?
**A:** Not yet. Cryptocurrency support is planned for future releases.

### Q: Do I need internet for currency conversion?
**A:** No! All conversions work offline using built-in rates.

### Q: Can I add custom currencies?
**A:** Not currently, but this feature is planned for advanced users.

### Q: How do I handle currency conversion fees?
**A:** Add the fee as a separate expense in the "Banking & Fees" category.

## Tips for Multi-Currency Users

### 1. **Consistent Recording**
- Record transactions in the actual currency used
- Don't manually convert before entering
- Let the app handle conversions

### 2. **Regular Updates**
- Update account balances when exchange rates change significantly
- Review total net worth monthly
- Adjust budgets for currency fluctuations

### 3. **Tax Planning**
- Track foreign income separately
- Note exchange rate on income date for tax purposes
- Use income categorization for tax filing

### 4. **Budget Wisely**
- Set budgets in INR for consistency
- Consider currency fluctuations
- Add buffer for exchange rate changes

### 5. **Smart Account Management**
- Group accounts by currency
- Monitor exchange rate trends
- Plan transfers during favorable rates

## Technical Details

### Currency Configuration
Located in: `nri-wallet/src/utils/currencies.js`

```javascript
export const CURRENCIES = {
  INR: { code: 'INR', symbol: '₹', exchangeRate: 1 },
  USD: { code: 'USD', symbol: '$', exchangeRate: 83 },
  THB: { code: 'THB', symbol: '฿', exchangeRate: 2.3 },
  // ... more currencies
};
```

### Conversion Functions
```javascript
// Convert any currency to INR
convertToINR(amount, fromCurrency)

// Convert INR to any currency
convertFromINR(amountInINR, toCurrency)

// Convert between any two currencies
convertCurrency(amount, fromCurrency, toCurrency)

// Format with currency symbol
formatCurrency(amount, currencyCode)
```
